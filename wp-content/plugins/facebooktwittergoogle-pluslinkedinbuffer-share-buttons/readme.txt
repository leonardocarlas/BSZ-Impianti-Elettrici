== Description ==
Simple Social Share Buttons - Facebook Like, Twitter, Google +, Buffer, LinkedIn, StumbleUpon, Pinterest Share Buttons after post contents.

<strong>Features:<strong>

* Smart Pinterest share, allows you to set the image or get the first image automatically

* Stylish Buffer button show.

* All buttons can show horizontal counts.

* Show buttons after post contents by default after activate this plugin.

Support: For more info or any problem, ask to <a href="http://freebloggingtricks.com/facebooktwittergoogle-plus-linkedin-buffer-share-buttons-wordpress/" target="_blank">Simple Social Share Buttons Support Blog</a>

Read more about the plugin at http://freebloggingtricks.com/facebooktwittergoogle-plus-linkedin-buffer-share-buttons-wordpress/?ref=wp
